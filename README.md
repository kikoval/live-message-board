Live blog
===

# Requirements

* PHP >=5.3
* [Composer](http://getcomposer.org/)

## Installation

1. Run `php composer.phar install`
2. Rename `config.php.dist` to `config.php` and set-up the values.
3. Run `php vendor/bin/doctrine orm:schema-tool:create` to initialize the database.
